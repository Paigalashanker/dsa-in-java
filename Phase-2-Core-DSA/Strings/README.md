# Phase 2 – Core DSA: Strings

This section contains beginner-friendly Java implementations for string-based problems from my **90-day DSA roadmap** (Days 8–10).

## Topics

- **Basic Operations**
  - Reversal
  - Palindrome Checking
  - Frequency Count

## How to run (Java)

Each program is standalone (no packages).

1. Open a terminal in the program folder.
2. Compile:
   ```bash
   javac FileName.java
   ```
3. Run:
   ```bash
   java ClassName
   ```

> Tip: Don't commit `.class` files to GitHub (see `.gitignore`).

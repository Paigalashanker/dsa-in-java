  - Insertion at Middle
  - Insertion at End

- **Array Deletion**
  - Deletion at Beginning
  - Deletion at Middle
  - Deletion at End

## How to run (Java)

Each program is standalone (no packages).

1. Open a terminal in the program folder
2. Compile:
   ```bash
   javac FileName.java
   ```
3. Run:
   ```bash
   java ClassName
   ```

> Tip: Don’t commit `.class` files to GitHub (see `.gitignore`).

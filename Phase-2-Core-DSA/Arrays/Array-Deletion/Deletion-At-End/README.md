# Deletion at End

## What it does

Removes the **last element** from the array by simply decreasing the element count.

## Algorithm

1. Read `n` elements into an array.
2. Decrease `n` by 1.
3. Print the updated array.

## Example

```
Input:  [10, 20, 30, 40, 50]
Output: [10, 20, 30, 40]
```

## Complexity

|           | Value |
| --------- | ----- |
| **Time**  | O(1)  |
| **Space** | O(1)  |

# Doubly Linked List

## Node Structure

Define Node with data, next pointer, and previous pointer.

## Advantages

- Bidirectional traversal
- Easier deletion (no need to find previous node)
- More memory usage than singly linked list

## Operations to Implement

1. Insert at beginning
2. Insert at end
3. Insert at position
4. Delete from beginning
5. Delete from end
6. Delete at position
7. Forward and backward traversal

## Implementation

Write complete doubly linked list implementation in `DoublyLinkedList.java`

# Days 15-20: Linked List

Master linked list data structures and related algorithms for interviews.

## Topics Covered

### 1. [Singly Linked List](Singly-Linked-List/)

- Node structure definition
- Insertion operations
- Deletion operations
- Traversal methods

**File:** `SinglyLinkedList.java`

### 2. [Doubly Linked List](Doubly-Linked-List/)

- Node structure with forward and backward pointers
- Bidirectional traversal
- Insertion and deletion
- Advantages over singly linked list

**File:** `DoublyLinkedList.java`

### 3. [Middle Element](Middle-Element/)

- Find middle element of linked list
- Two-pointer (slow and fast) technique
- Edge cases handling

**File:** `MiddleElement.java`

### 4. [Delete Loop Detection](Delete-Loop-Detection/)

- Floyd's Cycle Detection Algorithm
- Detect if cycle exists
- Length of cycle
- Remove cycle from linked list

**File:** `DeleteLoopDetection.java`

### 5. [Reverse Using Pointers](Reverse-Using-Pointers/)

- Reverse linked list iteratively
- Reverse using recursion
- Reverse in groups

**File:** `ReverseLinkedList.java`

### 6. [Dummy Node Technique](Dummy-Node-Technique/)

- Simplify edge case handling
- Remove duplicates
- Clean code patterns
- Best practices

**File:** `DummyNodeTechnique.java`

## Learning Objectives

- Understand pointer manipulation
- Master common linked list patterns
- Handle edge cases effectively
- Interview problem solving strategies

## Key Algorithms

- Two-pointer technique
- Floyd's cycle detection
- Dummy node pattern
- Recursive reversal

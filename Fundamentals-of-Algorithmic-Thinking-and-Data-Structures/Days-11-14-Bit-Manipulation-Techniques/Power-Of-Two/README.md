# Power of Two

## Problem Statement

Determine if a given number is a power of 2.

## Approaches

1. Using logarithm
2. Using bitmask: n & (n-1) == 0
3. Bit counting method

## Implementation

Write your solution in `PowerOfTwo.java`

## Examples

- 1, 2, 4, 8, 16, 32... are powers of 2
- 3, 5, 6, 7... are not powers of 2

# Big-O Notation

Big-O notation describes the upper bound of an algorithm's growth rate as the input size increases.

## Topics in this folder

- O1 Constant Time
- On Linear Time
- On2 Quadratic Time
- Ologn Logarithmic Time

## Why Big-O is useful

- It helps compare solutions quickly.
- It ignores machine-specific details.
- It focuses on how an algorithm scales.

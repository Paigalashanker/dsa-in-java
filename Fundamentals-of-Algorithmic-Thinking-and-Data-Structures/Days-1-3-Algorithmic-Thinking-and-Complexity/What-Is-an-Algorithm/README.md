# What Is an Algorithm

An algorithm is a finite set of clear steps used to solve a problem.

## Key points

- It takes some input.
- It performs well-defined steps.
- It produces an output.
- It must finish in finite time.

## Simple examples

- Find the largest number in an array.
- Check whether a string is a palindrome.
- Search for a value in a list.

## Why this matters

DSA is not only about code. It is about choosing the right sequence of steps to solve a problem correctly and efficiently.
